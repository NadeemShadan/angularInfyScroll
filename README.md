# angularInfyScroll
A simple infinty scroll for angularjs

NPM
```bash
$ npm i nginfyscroll
```
BOWER

```bash
$ bower i nginfyscroll
```

## How to use
Inject in app.js 
```js
angular
    .module('myApp',['infyScroll']);
```
pass in your function as the value of ng-infy-scroll with out parentheses

```html
<div ng-infy-scroll="loadMore">
```
